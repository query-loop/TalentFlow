


# TalentFlow

## How to Run TalentFlow
1. **Clone the repository:**
	```sh
	git clone https://github.com/query-loop/TalentFlow
	cd TalentFlow
	```
2. **Start all services (Docker Compose):**
	```sh
	docker-compose up --build
	```
3. **Access the UI:**
	- Open your browser at `http://localhost:3000` (default frontend port).
4. **Run experiments and scripts:**
	- See `tools/` for scripts and utilities.
